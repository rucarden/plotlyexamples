Your intention is to make the plots in Matlab, however much of what you want to do in plotly can be accomplished in javascript. The interaction between plotly and matlab focuses mainly on translating a matlab plot into a plotly graph. Some of these example will assume that there already exists a plotly that has been created by other means.
    Link to this page hosted in github: https://rucarden.github.io/plotlyexamples/
  • hello_world.html
    This example demonstrate how to listen for events from an embedded plotly graph. The ping message and pong event appear to be a good way of determining when a plotly graph has finished loading.
  • slider.html
    This example has two slider examples for the same plot. The first adjust the window for a moving average. In the second the slider changes the color of the plotted line.
  • streaming.m, streaming.html
